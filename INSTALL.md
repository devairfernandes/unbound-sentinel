# 📦 Instalação do Unbound Sentinel

## ⚠️ Instalação exclusiva via ISO Oficial

O Unbound Sentinel **não possui instalação manual pública**.

A distribuição é feita exclusivamente através de uma **ISO customizada** baseada em Rocky Linux 9.7, com todo o ambiente pré-configurado e otimizado.

## Como obter

Entre em contato para adquirir sua licença e receber o link da ISO oficial:

📧 **Email:** comercial@sentineldns.uk

## Para clientes existentes

Se você já possui o Sentinel instalado e precisa atualizar, utilize o botão de **Atualização OTA** dentro do próprio painel, ou entre em contato com o suporte.
