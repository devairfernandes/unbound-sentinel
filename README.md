# 🛡️ Unbound Sentinel

![Sentinel Header](https://raw.githubusercontent.com/devairfernandes/unbound-sentinel/main/assets/banner-v2.png)

**Unbound Sentinel** é um painel de monitoramento avançado para servidores DNS Unbound, projetado para oferecer visibilidade total, performance e segurança em tempo real. Com uma interface ultra-moderna e recursos de nível NOC (Network Operations Center), o Sentinel transforma dados brutos de DNS em inteligência visual.

---

## ✨ Recursos Principais

### 🚀 Performance & Monitoramento
- **Estatísticas em Tempo Real:** Acompanhe QPS (Queries Per Second), Latência e taxa de acerto de Cache instantaneamente.
- **Gráficos Dinâmicos:** Visualização histórica de tráfego de rede e carga de processamento.
- **Top Domains & Clients:** Identifique rapidamente os maiores consumidores e domínios mais acessados.

### 💎 Recursos Premium (Versão PRO)
- **🌍 Globo 3D de Tráfego:** Visualização geoespacial das requisições DNS em um globo interativo.
- **📺 Modo TV (NOC View):** Interface otimizada para grandes telas e centros de monitoramento, com atualização automática e layout expandido.
- **⚡ DNS Benchmark:** Compare a performance do seu servidor contra os principais players do mercado (Google, Cloudflare, Quad9).
- **🛠️ Editor de Configuração:** Modifique arquivos do Unbound diretamente pelo navegador com segurança.

## 🛠️ Instalação Rápida

O Sentinel foi projetado para ser instalado em menos de 2 minutos em qualquer servidor Linux (CentOS, Debian, Ubuntu, Rocky Linux).

```bash
# Baixe o pacote e execute o instalador
chmod +x install.sh
./install.sh
```

> Para instruções detalhadas, consulte o [Guia de Instalação](INSTALACAO.md).

## 🔒 Segurança & Privacidade

- **Acesso SSH Seguro:** O dashboard se conecta ao Unbound via túnel SSH criptografado.
- **Autenticação Robusta:** Proteção de login com credenciais configuráveis.
- **Zero Exposure:** Seus dados de configuração permanecem no seu servidor.
